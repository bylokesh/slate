# Slate - The Blank Page, Reimagined

Slate is a beautiful, local-first block editor built for speed, focus, and absolute privacy. Write at the speed of thought without the noise of modern software.

## Features

- **Blazing Fast Block Editor**: Notion-esque `/` commands for text, headings, lists, todos, code, quotes, and more.
- **Local-First & Private**: Zero cloud dependency. All your notes are stored securely on your device using IndexedDB. No accounts, no trackers. 
- **Typography & Customization**: Switch between Sans, Serif, and Mono fonts. Adjust page width and text sizing to match your reading and writing preferences.
- **Export Ready**: Seamlessly export your documents to beautifully formatted, high-resolution PDFs.
- **Fully Responsive**: Works beautifully across mobile, tablet, and desktop layouts.

## Tech Stack

- **Framework**: React 18 + Vite + TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Storage**: IndexedDB (idb)
- **Deployment**: Vercel ready

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
